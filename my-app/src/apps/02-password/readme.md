# [おみくじミニアプリ] - TypeScript リプレース・プロジェクト

## 1.アプリ概要

- **目的**: [ボタン操作によるランダムな結果の表示と、複数の状態管理の習得]
- **使用技術**: React, TypeScript, Tailwind CSS

## 2. 設計メモ

ボタンがクリックされるたびに、英字(大文字小文字どちらも)が格納された配列から、指定された長さの結果がランダムに返ってくる設計。これに加えてオプションとして数字と記号込みの結果出力も可能。CSSは上からパスワード出力欄(初期値もランダム生成)、ボタン、オプションとして長さ指定のスライダー、数字と記号それぞれにチェックボックスという構成である。なお、この長さは初期値は最小値8で、最大値は16であり、値をユーザーは視認可能。

### ① 機能要件

- ボタンをクリックして英字列(パスワード)を生成
- 結果をクリック毎にランダムに確認(ページの読み込み時も)
- オプションのスライダーで長さの指定可能(制限あり)
- 数字と記号を含めたパスワードの生成可能オプション

### ② 状態定義（useState）

| 状態変数名 | 型 | 初期値 | 役割 |
| :--- | :--- | :--- | :--- |
| `result` | `string` | `"？"` | 生成された結果を保持 |
| `length` | `number` | `8` | 長さの指定オプション |
| `useNumbers` | `boolean` | `false` | 数字の追加オプション |
| `useSymbols` | `boolean` | `false` | 記号の追加オプション |

### ③ 主要ロジック

- **抽選処理**: 配列からランダムなインデックスを算出するロジック。
