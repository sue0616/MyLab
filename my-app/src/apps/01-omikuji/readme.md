# [おみくじミニアプリ] - TypeScript リプレース・プロジェクト

## 1.アプリ概要

- **目的**: [ボタン操作によるランダムな結果の表示と、状態管理の習得]
- **使用技術**: React, TypeScript, Tailwind CSS

## 2. 設計メモ

ボタンがクリックされるたびに、文字列が格納された配列から、ランダムに結果が返ってくる簡素な設計。結果は大吉、中吉、小吉、凶の４つ。CSSはおみくじを模したものであり、上から今日の運勢という題名、運勢(初期値はクエスチョンマーク)、ボタンの構成である。

### ① 機能要件

- ボタンをクリックして運勢を引く
- 結果をクリック毎にランダムに確認

### ② 状態定義（useState）

| 状態変数名 | 型 | 初期値 | 役割 |
| :--- | :--- | :--- | :--- |
| `result` | `string` | `"？"` | 抽選された結果を保持 |

### ③ 主要ロジック

- **抽選処理**: 配列からランダムなインデックスを算出するロジック。
